CSS RGB Colors

An RGB color value represents RED, GREEN, and BLUE light sources. Try it yourself.

rgb(255, 0, 0)

rgb(0, 0, 255)

rgb(60, 179, 113)

rgb(238, 130, 238)

CSS HEX Colors

A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color. Try it yourself.

#ff0000

#0000ff

#3cb371

#ee82ee

CSS HSL Colors

HSL stands for hue, saturation, and lightness. Try it yourself.

hsl(0, 100%, 50%)0

hsl(240, 100%, 50%)

hsl(147, 50%, 47%)

hsl(300, 76%, 72%)
