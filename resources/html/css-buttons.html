Button Width

By default, the size of the button is determined by its text content (as wide as its content). Use the width property to change the width of a button: Tip:Use pixels if you want to set a fixed width and use percent for responsive buttons (e.g. 50% of its parent element). Resize the browser window to see the effect. Try it yourself.
